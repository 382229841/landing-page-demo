# Product Landing Page Demo

A modern SaaS landing page built with pure HTML/CSS.  
纯 HTML/CSS 构建的现代 SaaS 产品着陆页。

## ✨ Features / 功能

- 🎯 **Hero Section** — gradient headline, CTA buttons
- 🧩 **Feature Grid** — 6 feature cards with icons
- 📈 **Stats Bar** — social proof numbers
- 🎨 **CTA Section** — gradient background, conversion-focused
- 📱 **Fully Responsive** — mobile-first design
- ⚡ **Zero Dependencies** — no framework, no JS framework

## 🛠️ Tech Stack / 技术栈

- Semantic HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- No JavaScript framework
